Understanding the Equation of a Line

\(Y = \beta X + \epsilon\)

The Full Equation


\[Y = \beta X + \epsilon\]

Breaking It Down

\[\color{#4FC3F7}{Y} = \beta X + \epsilon\]

👁️ \(Y\) — The Outcome

\(Y\) is what we observe directly in the world.
It’s real, measurable, and varies across individuals or cases.
Think of it as the data we actually collect.

\[Y = \color{#FF8A65}{\beta} X + \epsilon\]

🔒 \(\beta\) — The Parameters (Unobserved, Indirect)

\(\beta\) is never directly seen — we can only estimate it from data.
It represents the true relationship between \(X\) and \(Y\) in the population.
We infer \(\beta\) indirectly through the data we collect.

\[Y = \beta X + \color{#CE93D8}{\epsilon}\]

🌫️ \(\epsilon\) — The Error (Unobserved Uncertainty)

\(\epsilon\) captures everything we didn’t measure or can’t explain.
It is unobserved — we never know it directly.
It represents the irreducible randomness in the world.

\(Y\) — Observed but Variable

\[\color{#4FC3F7}{Y} = \beta X + \epsilon\]


\(Y\) is observable.
We see it, measure it, collect it.

Examples: - Income of a person - Blood pressure of a patient
- Test score of a student


But \(Y\) is not fixed.
It varies — sometimes wildly — across observations.
That variation is why we need a model.

\(Y\) Has a Mean

\[\color{#4FC3F7}{E[Y]} = \beta X\]


The expected value (mean) of \(Y\) given \(X\) is a smooth function of \(X\).
This is the signal we’re trying to recover.


\(\beta\) — Unobserved, Indirect

\[Y = \color{#FF8A65}{\beta} X + \epsilon\]


\(\beta\) is the true population parameter.
It exists in theory — but we never observe it directly.


We only ever have estimates \(\hat{\beta}\) computed from data.

\[\hat{\beta} = (X'X)^{-1}X'Y\]

These estimates are indirect — they depend entirely on what we observed in \(Y\) and \(X\).


::: {.callout-warning icon=false} ### The Estimation Gap True \(\beta\) ≠ Estimated \(\hat{\beta}\)
The difference is sampling uncertainty.

:::

\(\epsilon\) — Unobserved Uncertainty

\[Y = \beta X + \color{#CE93D8}{\epsilon}\]


\(\epsilon\) is the error term — permanently unobserved.

It captures everything that \(X\) doesn’t explain: - Measurement error
- Omitted variables
- Inherent randomness in the world


We assume \(\epsilon \sim N(0, \sigma^2)\)

  • Mean zero: no systematic bias
  • Finite variance: uncertainty is bounded
  • Independent of \(X\): errors aren’t predictable

Putting It All Together


\[\color{#4FC3F7}{Y} = \color{#FF8A65}{\beta} X + \color{#CE93D8}{\epsilon}\]


Term Color Observed? Nature
\(Y\) Blue ✅ Yes — directly Outcome; has a mean
\(\beta\) Orange ❌ No — indirectly Population parameter; estimated
\(\epsilon\) Purple ❌ No — never Random uncertainty

The Key Insight



We observe \(Y\)

We estimate \(\beta\)

We assume \(\epsilon\)


Statistics is the art of learning about the unobserved (\(\beta\), \(\epsilon\)) from the observed (\(Y\)).
